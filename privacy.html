New API

AI API 网关服务 · stevetoken.kai.com

生效日期：2026 年 7 月 10 日

1. 引言

New API（stevetoken.kai.com，以下简称"我们"）深知个人信息对您的重要性。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。使用本服务即表示您同意本隐私政策所述的数据处理方式。

2. 我们收集的信息

我们遵循数据最小化原则，仅收集为提供 API 网关服务所必需的信息：

数据类别具体内容收集目的
账户信息 电子邮箱、用户名、密码哈希 账户创建与身份识别
交易记录 充值金额、支付渠道、时间戳 财务核算与账单查询
API 调用元数据 请求量、模型标识、Token 消耗量 用量统计与计费
技术日志 IP 地址、请求时间戳、User-Agent 安全防护与故障排查

我们不会收集 API 请求体内容（prompt 与 completion）。您的对话内容仅通过我们的网关实时转发至后端模型提供商，不在我们服务器上落地存储。

3. 信息的使用

我们收集的信息仅用于以下用途：

  • 服务提供：验证身份、处理充值、统计用量、路由 API 请求。
  • 安全保障：检测和防止欺诈、滥用及未授权访问。
  • 客户支持：响应用户的咨询、投诉及账户相关问题。
  • 服务改进：基于聚合的匿名统计数据优化系统性能。

我们不会将您的个人信息用于自动化决策或用户画像，也不会将其出售给第三方。

4. 信息的共享与披露

我们仅在以下情形中与第三方共享您的信息：

  • 后端模型提供商：当您发起 API 请求时，您的 prompt 内容将实时转发给对应的模型提供商（如 OpenAI、Anthropic 等），以完成模型推理。这些提供商有各自的隐私与数据处理政策。
  • 支付服务商：处理您的充值付款时，交易信息将传递给支付网关完成结算。
  • 法律要求：依法向司法或行政机构提供必要的账户信息。

除此之外，我们不会向其他任何第三方出售、出租或分享您的个人信息。

5. 数据存储与安全

您的账户信息与交易记录存储在加密数据库中。我们采用行业标准的安全措施保护您的数据，包括传输层加密（TLS）、访问控制及定期安全审计。

  • API 请求日志（仅含元数据，不含对话内容）保留不超过 24 小时，用于故障排查与滥用监控。
  • 账户数据在账户存续期间保留。账户注销后，个人信息将在 30 天内删除（法律法规另有要求的除外）。

6. 您的权利

根据适用法律，您对自身信息享有以下权利：

  • 访问与导出：您可随时登录控制台查看余额、调用明细及基本账户信息。
  • 更正：如账户信息有误，您可通过控制台自行修改或联系客服协助。
  • 删除：您可申请注销账户，我们将在验证身份后删除您的个人信息。
  • 撤回同意：您可随时停止使用本服务。对于非必要的 Cookie 或跟踪（本服务不使用），您有权拒绝。

7. Cookie 与跟踪技术

New API 网站本身不使用任何第三方跟踪脚本或广告 Cookie。我们仅使用会话必需的 Cookie 维护登录状态，这些 Cookie 在您登出或关闭浏览器后失效。

8. 未成年人保护

本服务不面向 18 周岁以下的未成年人。我们不会故意收集未成年人的个人信息。如发现误收集，将立即删除。

9. 政策更新

我们可能不时更新本隐私政策。重大变更将通过邮件或站内通知告知。更新后的政策发布于本页面，届时生效日期将更新。建议您定期查阅。

10. 联系方式

如对本隐私政策有任何疑问、行使您的数据权利或报告安全问题，请通过以下方式联系我们：

  • 邮箱：support@kai.com
  • 官网：stevetoken.kai.com

我们将在 5 个工作日内回复您的请求。
