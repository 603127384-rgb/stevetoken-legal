可接受使用政策

最后更新：2026年7月

本可接受使用政策（"AUP"）规定了您在使用 Kai-team API（"本服务"）时的行为规范。使用本服务即表示您同意遵守本政策。

1. 禁止用途

您不得将本服务用于以下任何用途：

1.1 违法活动

  • 违反任何适用法律、法规或政府命令的活动
  • 欺诈、洗钱、恐怖主义融资或其他金融犯罪
  • 未经授权访问他人计算机系统或网络

1.2 有害内容生成

  • 生成或传播儿童性虐待材料（CSAM）或任何形式的儿童剥削内容
  • 生成或传播仇恨言论、骚扰、欺凌或歧视性内容
  • 生成或传播极端暴力、酷刑或血腥内容
  • 促进自残、自杀或危险行为

1.3 隐私与安全侵犯

  • 未经同意收集、存储或分享他人个人信息
  • 社会工程攻击、钓鱼、诈骗或身份盗窃
  • 生成深度伪造内容用于欺骗、诽谤或骚扰他人
  • 监控或跟踪他人未经其知情同意

1.4 恶意技术用途

  • 生成恶意软件、勒索软件、病毒、蠕虫或其他恶意代码
  • 进行拒绝服务攻击（DDoS）或网络攻击
  • 规避安全措施或访问控制
  • 大规模自动注册虚假账户或刷量行为

1.5 知识产权侵权

  • 大规模生成侵犯他人版权、商标或专利的内容
  • 未经授权复制或分发受版权保护的材料
  • 以侵犯知识产权为目的的训练或微调模型

1.6 其他禁止行为

  • 转售或重新分发 API 密钥（除非与平台有明确书面协议）
  • 通过自动化方式规避速率限制或配额限制
  • 对服务进行逆向工程、反编译或提取模型参数
  • 将输出冒充人类创作内容以误导他人

2. 内容安全要求

您在使用本服务时应确保：

  • AI 生成的内容不违反上述禁止用途
  • 对用户输入实施适当的内容过滤
  • 不将本服务嵌入可能违反本政策的应用程序或服务中

3. 违规处理

如发现您违反本政策，我们保留以下权利：

  • 立即暂停或终止您的 API 访问权限
  • 不退还已支付的费用
  • 在法律要求的情况下向有关当局报告违规行为
  • 保留追究法律责任的权利

4. 举报

如发现违反本政策的行为或内容，请通过以下方式举报：

📧 abuse@kai.com — 我们将在 48 小时内处理。

5. 政策更新

我们可能会不时更新本政策。重大变更将通过服务公告或邮件通知。继续使用本服务即表示接受更新后的政策。
