1. 服务概述

New API（以下简称"本服务"）是由 stevetoken.kai.com 运营的 AI API 网关平台。本服务向注册用户提供统一的 API Token 购买与管理服务，用户可通过本平台获取对主流大语言模型（LLM）API 的调用额度。

本服务本质为 API Token 分销与网关代理：用户购买 Token 额度后，通过本平台提供的统一端点发起 API 请求，本平台负责将请求路由至后端模型提供商并返回响应。用户无需分别对接各个模型提供商的独立接口。

  • 用户须年满 18 周岁或在其所在司法管辖区达到法定签约年龄。
  • 注册即表示您同意本条款的全部内容。如不同意，请勿使用本服务。

本服务采用预付费充值模式：用户通过支持的支付渠道购买 API 额度（以 Token 或调用次数计），实际消耗按各模型的官方定价折算扣除。

  • 充值：充值金额即时到账，不可转让，不可兑换为现金。
  • 退款：一般情况下，已充值的额度不予退款。以下情况除外：
    （a）因本服务系统故障导致额度未到账且超过 72 小时未恢复；
    （b）本服务主动终止运营，将按剩余未消耗额度比例退还。
